System requirments for R:
* a minimum of 1GB of RAM
* a non-root user with sudo privileges. 


1. Open a terminal window and set the directory in your terminal to the installer's folder path.

2. If you have already R installed skip this step, if it is not installed run the following commands:
----------------------------------
chmod +x All_installer.sh 
./All_installer.sh 
----------------------------------

3. If you have already R installed already, run this:
----------------------------------
chmod +x 2_R_lib_install_linux.sh 
./2_R_lib_install_linux.sh 
----------------------------------

4. Run runACAQ_LINUX.sh
